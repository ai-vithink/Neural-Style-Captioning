## Prerequisites
* python3
### Installation
```sh
 git clone https://github.com/shan1322/Neural-Style-Captioning
 cd Neural Style Captioning
 pip3 install -r requirements.txt
 ```
 ### Running
```sh
 run cap.ipynb
```
